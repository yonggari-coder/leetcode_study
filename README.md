# leetcode_study 📚

> LeetCode 풀이 + 문제별 회고를 모은 학습 저장소. **푼 것보다 "왜 그렇게 풀었는지"를 남기는 것**에 무게를 둡니다.

## 🗂 구성

```
leetcode_study/
├── Problem_list/                       # 일반 문제 풀이 + write-up
│   ├── 00009_Pelindrome_Number.md      # 풀이 회고
│   ├── 01559_Detect_Cycles_In_2D_Grid.md
│   ├── 2833_Furtest_Point_From_Origin.cpp
│   └── two_sum.cpp
└── Interview_Instance_1/               # 인터뷰 대비 단골 문제 별도 트랙
    └── LRU/
```

- **`.md` write-up과 `.cpp` 풀이를 함께**: 처음 본 문제의 사고 과정을 마크다운에 남기고, 풀이 코드를 같이 커밋합니다.
- **`Interview_Instance_1`**: 캐시 구현(LRU) 등 인터뷰 단골 자료구조를 별도 폴더에 모아 빠르게 다시 볼 수 있게.

## 🛠 언어

- **C++** (STL 컨테이너, 입출력 최적화)

## 🎯 이 repo의 의도

- **풀이를 자랑하기 위함이 아닌, 미래의 나에게 보여주기 위함**: 같은 유형을 다시 만났을 때 빠르게 떠올리도록 정리합니다.
- **마크다운 회고 습관**: 문제 번호 prefix(`00009_`, `01559_`)로 정렬을 일관되게 유지.
